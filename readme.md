# Image Gallery Component

A responsive image gallery built with StencilJS.

## Live Demo

View it here: [https://amaasik.github.io/gallery-project/](https://amaasik.github.io/gallery-project/)

## How to Run Locally

### 1. Clone the project

```bash
git clone https://github.com/amaasik/gallery-project.git
cd gallery-project
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

The project will open in your browser at `http://localhost:3333`

## Features

- Responsive design (mobile & desktop)
- Lightbox modal for desktop
- Keyboard navigation (Arrow keys, Escape)
- Orange hover effects
- Thumbnail navigation

## Commands

- `npm start` - Start development server
- `npm run build` - Build for production
